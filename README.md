# ml_courses

## ama (Aprendizagem de Máquina)

Modelos clássicos e introdutórios de ML

- Lista 1: Regressão Linear.
- Lista 2: Regressão Logística, Naive Bayes Gaussiano, Análise de Discriminante Gaussiano e Regressão Softmax.
- Lista 3: Árvore de Decisão (Scikit Learn) e KNN.
- Lista 4: Multi Layer Perceptron (Scikit Learn).
- Lista 5: SVM e Random Forest (Scikit Learn).
- Lista 6: K-Médias e PCA.

Trabalho Final: Classificação de potabilidade de amostras de água. 

## ama_prob (Aprendizagem de Máquina Probabilístico)

Modelos probabilísticos de ML, sendo alguns bayesianos

Listas da disciplina de Aprendizagem de Máquina Probabilística

- Lista 1: Bernouli Naive Bayes Gaussiano.
- Lista 2: Regressão Linear Bayesiana, Regressão Polinomial Bayesiana, Regressão Logística IRLS e Regressão Logística Bayesiana.
- Lista 3: GMM.
- Lista 4: PPCA com algoritmo EM e Inferência Variacional para uma gaussiana univariada.
- Lista 5: Processos Gaussianos para Regressão.
- Lista 6: Autoencoder e Variational Autoencoder, padrão e condicional. 

Trabalho Final: Geração de músicas de NES utilizando modelos probabilísticos.

## adv_topics_ama (Tópicos Avançados em Aprendizagem de Máquina)

Modelos baseados em aprendizado profundo 

- Lista 1: Implementação Numpy do MLP usando SGD, Regressão simbólica.

- Lista 2: Convergência de modelos, NoisyGD, SGD com momentum e ADAM. Implementação de RNNs, LSTM, GRU e RNN residual.

Trabalho Final: Implementação de transformer e comparativo de desempenho com LSTM para o problema de autômatos celulares.
